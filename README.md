# Spanish Translator AI Training landing page

Landing page for validating the Spanish Translator AI Training course.

## Live assets

- `index.html` — landing page in Spanish
- `Como-Prosperar-Traductor-Era-IA.pdf` — lead magnet PDF
- `lead-magnet.md` — source content for the PDF

## Signup tracking

The waitlist form posts to FormSubmit and sends each signup to `ekko-outreach@agentmail.to`.

Captured fields:

- name
- email
- course_waitlist
- lead_magnet
- pdf_url
- tracking_page
- tracking_source
- tracking_medium
- tracking_campaign
- tracking_referrer
- submitted_at

Use UTM parameters for campaign attribution, for example:

`/?utm_source=facebook&utm_medium=community&utm_campaign=translator_ai_validation`
